# Postter

CRUD-APP + Google SignIn 

## Installation

Use the package manager [npm](https://www.npmjs.com/) to install App.

```bash
npm install npm
git clone https://github.com/Kasmadei/Postter
```
Tech Stack:
- HTML
- CSS (Bootstrap)
- JavaScript (React, Redux)
